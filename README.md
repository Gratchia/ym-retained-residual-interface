## Repository purpose

This repository contains the public reproducibility package for the manuscript:

*A Bianchi-Compatible Retained/Residual Interface for Finite Lattice Yang--Mills Theory*

Target journal preparation: SciPost Physics Core.

## Scientific scope

This repository supports finite-regulator structural checks only, including:

- certified cuboidal-template Bianchi compatibility;
- finite word-checking for certified Bianchi templates;
- retained/residual interface documentation;
- conditional reflection-admissible stem checking mechanics.

It does **not** prove:

- continuum Yang--Mills existence;
- infinite-volume limit;
- Osterwalder--Schrader reconstruction;
- Hamiltonian construction;
- transfer decay;
- confinement;
- spectral mass gap;
- a complete RG/blocking map.

## Reflection-stem caveat

The included reflection-stem certificate is illustrative unless project-specific retained-template certificates are later supplied. Reflection-admissibility remains an explicit conditional assumption in the manuscript.
